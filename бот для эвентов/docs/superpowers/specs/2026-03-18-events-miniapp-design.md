# Telegram Mini App — События Москвы

**Дата:** 2026-03-18
**Статус:** Approved

---

## Обзор

Telegram Mini App для профессиональной аудитории Москвы (маркетологи, аналитики, продажники, AI-специалисты). Агрегирует актуальные мероприятия — конференции, нетворкинги, вебинары, воркшопы — из нескольких источников. Пользователь открывает приложение сам (pull-модель), фильтрует по интересам и сохраняет понравившиеся события.

**Только Москва. MVP.**

---

## Архитектура

```
[Telegram Bot + Mini App]
        ↓
[Telegram Web App SDK] → фронт (HTML/JS/CSS)
        ↓
[Supabase REST API] ← автогенерируется из схемы БД
        ↓
[PostgreSQL в Supabase]
        ↑
[Python парсер] — запускается раз в день (GitHub Actions cron)
        ↑
[Supabase Studio] — ручная модерация и добавление событий
```

**Компоненты:**
- `bot.py` — Telegram бот, выдаёт ссылку на Mini App
- `parser/` — Python скрипты парсинга по источникам
- `miniapp/` — фронт: HTML + Vanilla JS + Telegram Web App SDK
- Supabase — управляемый PostgreSQL + REST API + Studio (облако)

## Модель данных

### Таблица `events`

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | uuid | PK |
| `title` | text | Название события |
| `description` | text | Краткое описание |
| `category` | enum | `marketing`, `sales`, `analytics`, `ai`, `culture`, `other` |
| `format` | enum | `online`, `offline`, `hybrid` |
| `price_type` | enum | `free`, `paid` |
| `price_amount` | int | Цена в рублях (0 если free) |
| `event_date` | timestamptz | Дата и время начала |
| `speakers` | text[] | Список спикеров |
| `url` | text | Ссылка на регистрацию |
| `cover_image_url` | text | og:image с сайта мероприятия |
| `source` | text | Источник: `timepad`, `afisha`, `yandex_afisha`, `telegram`, `instagram`, `manual` |
| `is_approved` | bool | Прошло модерацию (default: false) |
| `created_at` | timestamptz | Дата добавления |

### Таблица `saved_events`

| Поле | Тип | Описание |
|------|-----|----------|
| `user_id` | bigint | Telegram user ID |
| `event_id` | uuid | FK → events.id |
| `saved_at` | timestamptz | Дата сохранения |

## UI/UX

### Главный экран — лента с обложками

- **Шапка:** заголовок "События Москвы" + кнопка фильтров (открывает bottom sheet)
- **Чипы категорий:** горизонтальный скролл (Все / 🤖 ИИ / 📈 Маркетинг / 💼 Продажи / 📊 Аналитика / 🎭 Культура)
- **Карточки событий** (вертикальная лента):
  - Обложка мероприятия (og:image, высота ~140px) с градиентом снизу
  - Бейджи на картинке: категория + цена/бесплатно
  - Название события (bold)
  - Краткое описание + спикеры
  - Дата, время, формат
  - Кнопка "Перейти" + иконка закладки
- **Нижний таб-бар:** 🗓 События | 🔖 Сохранённые

### Фильтры (bottom sheet)

Открывается свайпом или кнопкой. Секции:
- **Категория** (мультиселект): Все, ИИ, Маркетинг, Продажи, Аналитика, Культура, Другое
- **Формат:** Онлайн / Офлайн / Гибрид
- **Цена:** Любая / Бесплатно / Платно
- **Дата:** Эта неделя / Этот месяц / Выбрать диапазон
- Кнопки: "Сбросить" / "Применить"

### Экран сохранённых

Такая же карточная лента, но только события с `saved_events` текущего пользователя. Пустое состояние: "Ещё ничего не сохранено — найди что-то интересное 👆"

### Детальный экран события (опционально в MVP)

По тапу на карточку — открывается шторка снизу с полной информацией: описание, спикеры, дата, адрес, цена, кнопка "Зарегистрироваться" (переход по url).

## Источники данных и парсинг

| Источник | Метод | Стабильность |
|----------|-------|--------------|
| **Timepad** | REST API (бесплатный) | ✅ Высокая |
| **Афиша** (afisha.ru) | Парсинг HTML | 🟡 Средняя |
| **Яндекс Афиша** (afisha.yandex.ru) | Парсинг HTML | 🟡 Средняя |
| **Telegram-каналы** | Telegram Bot API (публичные каналы) | ✅ Высокая |
| **Instagram** | Instaloader или аналог | 🔴 Нестабильная (fallback) |

**Расписание:** GitHub Actions cron — ежедневно в 07:00 МСК.

**Обложки:** парсер вытаскивает `og:image` с URL каждого события при добавлении.

**Модерация:** новые события создаются с `is_approved = false`. Администратор одобряет через Supabase Studio. В ленту попадают только одобренные.

## Что не входит в MVP

- Push-уведомления о новых событиях
- Регистрация организаторов / самостоятельная подача событий
- Рейтинги и отзывы
- Карта событий
- Поиск по тексту
- Рекомендации на основе истории
